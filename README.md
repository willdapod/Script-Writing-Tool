# ScriptWriter Pro

A modern, distraction-free screenwriting tool. Format your scripts automatically with fountain-lite logic, scene detection, and local auto-save.

## Features
- **Distraction-Free Editor**: Clean, dark interface with glassmorphism.
- **Smart Formatting**: Auto-detects Scene Headings, Characters, and Transitions.
- **Focus Mode**: Hides the sidebar for pure writing.
- **Auto-Save**: Saves to browser local storage automatically.
- **Portable**: Single HTML file, runs offline in any browser.

## Quick Start
1.  **Standalone App**: Double-click `ScriptWriterApp.exe` to run the tool in its own window.
    > [!NOTE]
    > Requires [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) to be installed on Windows.
2.  **Web Pointer**: Alternatively, open `ScriptWriter.html` in any modern web browser.
3.  Start writing!

## Shortcuts
- **Tab**: Cycle between styling (Action -> Character -> Dialogue).
- **Esc**: Toggle **Focus Mode** / **Settings**.
- **Enter**: Create a new line.
- **Save**: Changes are automatically saved to your browser's local storage.
- **Export**: Click the download icon to save as a `.fountain` file.

## ⚙️ Settings & Accessibility
Click the **Cog Icon** in the header to access accessibility options:
- **Font Size**: Adjust the editor text size from Small to Huge.
- **High Contrast**: Switch to a pure black & white theme for maximum visibility.
- **Dyslexia Friendly**: Toggle a readable sans-serif font.
- **Reduced Motion**: Disable all UI animations.

All settings are automatically saved.
